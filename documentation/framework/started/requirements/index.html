Minimum System Requirements

The following HTML5 browser functions are required for the framework to run:

  • Support / Access to localStorage - please note that private browsing prevents access

Tested Support Matrix for Framework

Browser Operating System Version Support
Firefox MacOSX Mavericks 33.1 Works from Server and Desktop
Chrome MacOSX Mavericks 38.0.2125.122 Only Works from Server
Safari MacOSX Mavericks 7.0.5 Only Works from Server

If you want to run the framework locally but are having trouble due to not having your own web-server, we would recommend XAMPP or MAMP. These solutions provide a relatively easy way to set-up a server locally on your desktop device, but should not be used for production purposes.

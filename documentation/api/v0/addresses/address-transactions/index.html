address Transactions

GET <API_ENDPOINT>/v0/<CHAIN>/address/transactions/<ADDRESS>

Returns a Address object and an array of Transaction objects for the Address specified by the <CHAIN> and <ADDRESS> values.

By default, only minimal Address Transaction information is returned. For additional Transaction details use the ?showtxnio=1 query parameters.
